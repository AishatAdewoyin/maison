## Project Overview

Maison of Aisha is a premium e-commerce platform offering timeless elegance and high-quality fashion accessories for women, children, and mothers. It is designed to provide customers with an exceptional shopping experience while enabling administrators to efficiently manage the backend processes such as products, orders, and customer interactions.

---

## Features

### **Admin Functionalities**
1. **Admin Dashboard**:
   - Comprehensive view of total sales, orders, customers, and inventory.
2. **Product Management**:
   - Add, edit, delete, and categorize products.
   - Manage product images, descriptions, pricing, and stock levels.
3. **Order Management**:
   - View, process, and update order statuses (e.g., Pending, Shipped, Delivered).
   - Generate sales and order history reports.
4. **Customer Management**:
   - View and manage customer accounts and order history.
   - Handle customer feedback and inquiries.
5. **Promotions & Discounts**:
   - Set promotional offers, discount codes, and seasonal campaigns.
6. **Notification Management**:
   - Configure and send automated notifications for order updates, promotions, and other events.

---

### **Customer Functionalities**
1. **User-Friendly Navigation**:
   - Explore products by categories, featured collections, or search filters.
2. **Product Display**:
   - View detailed pages for products including images, descriptions, pricing, and customer reviews.
3. **Shopping Cart**:
   - Add or remove items, apply discounts, and view an updated cart summary.
4. **Checkout**:
   - Secure multi-step checkout process with various payment options (credit card, PayPal, Paystack).
5. **Customer Account**:
   - Register/login via email or social media, track orders, and update profile details.
6. **Feedback & Reviews**:
   - Submit reviews and ratings for products to help others make informed decisions.
7. **Wishlist**:
   - Save favorite items for future purchases.

---

### **Product Upload Functionalities**
1. **Bulk Upload**:
   - Admins can upload multiple products using CSV/Excel files.
2. **Image Upload**:
   - Drag-and-drop or file upload for product images.
3. **SEO Features**:
   - Add SEO-friendly metadata for improved search engine visibility.
4. **Stock Alerts**:
   - Set low-stock alerts and restock notifications for products.

## Technologies

### **Frontend**
- **HTML, CSS, JavaScript**: For core web structure and interactivity.
- **CSS Framework**: Bootstrap or Tailwind CSS for responsive design.
- **JavaScript Framework**: React.js for dynamic and scalable user interfaces.
- **State Management**: Redux or Context API for seamless frontend state handling.

### **Backend**
- **Programming Language**: Python with Django or Flask as the backend framework.
- **Database**: PostgreSQL for structured, reliable data storage.
- **API**: RESTful APIs or GraphQL for robust communication between frontend and backend.

### **Authentication**
- **Authentication System**: Django Authentication or Firebase for secure user logins.
- **Payment Gateway**: Stripe, Paystack, or PayPal for secure online transactions.

### **Hosting**
- **Frontend Hosting**: Netlify or Vercel for hosting the React frontend.
- **Backend Hosting**: Render, Heroku, or AWS for backend services.
- **Database Hosting**: AWS RDS or Heroku Postgres for scalable databases.

### **Version Control**
- **Git**: For version tracking, hosted on GitHub.

## Installation

### Prerequisites
1. Install **Node.js** for frontend dependencies.
2. Install **Python** for backend development.
3. Install **PostgreSQL** for the database.

### Steps
1. Clone the repository:
   ```
   git clone https://github.com/aishatadewoyin/maison-of-aisha.git
   cd maison-of-aisha
   
2. Set up the backend:
   
   ```
    cd backend
    python -m venv venv
    source venv/bin/activate  # For Windows: venv\Scripts\activate
    pip install -r requirements.txt
    python manage.py migrate
    python manage.py runserver

2. Set up the frontend:
   
   ```
   cd frontend
    npm install
    npm start

## Usage
1. To start the application:
    npm start
2. For development mode:
    npm run dev
